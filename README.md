# 🚀 SkillSwap

A simple platform where people can **share skills, learn skills, and connect with others**.

You can:
- Add a skill you know (Teach)
- Add a skill you want to learn
- Search skills easily
- Filter by Teach / Learn
- Delete your skills anytime

## 💡 What this project does

SkillSwap helps people:
- Share what they know
- Find what they want to learn
- Connect skills with others easily

Think of it as a **mini skill-sharing network**.

---

## ⚙️ How it works

1. Sign up
2. Log in
3. Get your dashboard
4. Add your skills
5. Search or filter skills
6. Manage (delete) anytime

---

## 🔐 Login System

- Secure signup/login system
